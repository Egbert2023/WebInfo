Cookie-Richtlinie
In dieser Anwendung werden keine Cookies verwendet.
Das bedeutet, dass weder so genannte "technisch notwendige Cookies" noch "Tracking-Cookies" auf dieser Webseite gesetzt oder gelesen werden.

Externe Links werden rot, fett und unterstrichen gekennzeichnet. z.B.: www.externerLink.de Wenn Sie auf so einen externen Link klicken, der die eigene Web-Anwendung verlässt wird es in den meisten Fällen passieren, dass die jeweiligen Anbieter ihrerseits Cookies verwenden. Darauf hat der Webseitenbetreiber keinen Einfluss. Bitte beachten Sie, dass Sie dann die entsprechenden Cookie-Banner erhalten und bearbeiten können, um sie ggf. ihren Bedürfnissen anzupassen um ihre Privatspäre zu schützen.

Wie kann man prüfen, ob keine Cookies gesetzt wurden?

Crome-Browser

  • Rufen Sie die zu prüfende Seite auf (am Beispiel: www.aleksander.de)
  • Betätigen Sie die F12-Taste
  • In dem sich neu geöffneten Fenster clicken Sie auf den Tab: "Application"
  • Dann wählen Sie auf der linken Seite den Bereich "Storage"
  • In dem Bereich Storage gibt es den Unterpunkt: "Cookies". Wählen sie diesen aus.
  • Jetzt können Sie die zu prüfende URL auswählen und es erscheint eine Übersicht über vorhandene Cookies.

Firefox-Browser

  • Rufen Sie die zu prüfende Seite auf (am Beispiel: www.aleksander.de)
  • Betätigen Sie die F12-Taste
  • In dem sich neu geöffneten Fenster clicken Sie auf den Tab: "Web-Speicher"
  • Dann wählen Sie auf der linken Seite den Bereich "Cookies". Wählen sie diesen aus.
  • Jetzt können Sie die zu prüfende URL auswählen und es erscheint eine Übersicht über vorhandene Cookies.
